# Changelog

All notable changes to this project will be documented in this file.

The format is inspired by [Keep a Changelog](https://keepachangelog.com/en/1.1.0/)
and the project uses semantic versioning.

## [0.5.0] - 2026-03-22

### Added

- Unified `SkillCompiler` facade for simpler framework-specific integration entrypoints
- OpenAI Agents streaming integration
- Google ADK callback bundle
- CrewAI listener integration
- LangChain callback handler integration
- LangGraph callback handler integration
- LlamaIndex callback handler integration
- Open-source repository files including license, contributing guide, code of conduct, security policy, roadmap, issue templates, PR template, CODEOWNERS, and GitHub Actions workflows

### Maintenance

- Repositioned the repository as the standalone SDK for the wider AI Skills Compiler platform
- Expanded README guidance for support levels, PyPI packaging, collaboration, and trusted publishing
- Updated package metadata and repository links for the public GitHub SDK repository

## [0.4.3] - 2026-03-21

### Changed

- Improved generic tracing behavior and framework helper exports before open-source packaging
